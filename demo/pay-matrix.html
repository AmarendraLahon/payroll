7th CPC Pay Matrix §03

Rows = Pay Levels, columns = Cells (annual increment stages). Each cell ≈ previous × 1.03, rounded up to the next ₹100. Basic Pay is read from the cell, never stored.

Highlighted cells: Level 4 / Cell 3 = ₹25,500 is Mr. Das's current position. Level 10 / Cell 1 = ₹56,100 is Mr. Roy's (retiring). Increment = move one cell right; promotion = jump to a higher Level with re-fixation.
